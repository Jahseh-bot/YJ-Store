## 一、产品概述

### 1.1 产品定位

YJ商城后台管理系统是面向商家和平台运营人员的B端管理系统，用于支撑电商平台的日常运营工作。分为商家后台（卖家中心）和运营后台（平台管理）两大部分。

### 1.2 系统架构

```
YJ商城后台管理系统
├── 商家后台（卖家中心）
│   ├── 工作台/首页
│   ├── 商品管理
│   ├── 订单管理
│   ├── 售后管理
│   ├── 客服管理
│   ├── 财务管理
│   ├── 店铺管理
│   ├── 数据中心
│   └── 账户设置
└── 运营后台（平台管理）
    ├── 工作台
    ├── 用户管理
    ├── 商家管理
    ├── 类目管理
    ├── 商品审核
    ├── 活动管理
    ├── 优惠券管理
    ├── 广告管理
    ├── 客服管理
    ├── 数据报表
    └── 系统设置
```

## 二、商家后台 - 工作台

### 2.1 经营概览

#### 交互说明

| 元素 | 交互行为 |
|------|----------|
| 今日数据卡片 | 显示今日订单数、今日销售额、今日访客数、今日转化率，大字显示数值，右侧显示较昨日涨跌(红色上箭头/绿色下箭头) |
| 数据刷新 | 点击"刷新"图标刷新数据，显示loading动画 |
| 时间筛选 | 默认显示"今日"，可切换"昨日"、"近7天"、"近30天"、"本年" |
| 订单提醒 | 显示待处理订单数量红色角标，点击跳转对应列表 |
| 异常预警 | 显示商品库存不足、违规预警等异常情况，点击跳转处理 |

### 2.2 待办事项

#### 交互说明

| 元素 | 交互行为 |
|------|----------|
| 待付款订单 | 显示数量，点击跳转订单列表(筛选待付款) |
| 待发货订单 | 显示数量，点击跳转订单列表(筛选待发货)，点击"批量发货"可批量操作 |
| 退货/售后单 | 显示数量，点击跳转售后列表，点击可查看详情并处理 |
| 商品违规 | 显示违规商品数量，点击查看违规原因和申诉入口 |
| 店铺评分 | 显示当前店铺综合评分，点击查看评分详情 |
| 催付款 | 对超时未付款订单点击"催一下"发送短信提醒 |

### 2.3 快捷入口

| 元素 | 交互行为 |
|------|----------|
| 发布商品 | 点击跳转商品发布页 |
| 处理订单 | 点击跳转订单列表 |
| 查看数据 | 点击跳转数据中心 |
| 店铺设置 | 点击跳转店铺设置页 |
| 常用功能 | 可自定义显示最多8个常用功能入口 |

## 三、商家后台 - 商品管理

### 3.1 商品列表

#### 交互说明

| 元素 | 交互行为 |
|------|----------|
| 商品搜索 | 输入商品名称/ID搜索，回车或点击搜索按钮执行搜索 |
| 状态筛选 | Tab切换：全部/上架/下架/待审核/违规，点击切换筛选 |
| 类目筛选 | 选择一级、二级类目，联动筛选 |
| 价格区间 | 输入最低价和最高价，点击"确定"筛选 |
| 批量操作 | 勾选多个商品后，顶部显示批量操作栏：批量上架/下架/删除/移动类目 |
| 商品表格 | 显示商品图片(缩略图)、商品名称/SKU、SKU编码、价格、库存、销量、状态、操作 |
| 商品图片 | 鼠标hover显示大图预览，点击跳转商品详情页 |
| 商品名称 | 显示商品标题，hover显示完整名称(超出省略)，点击跳转商品详情页 |
| SKU信息 | 显示SKU属性组合，如"颜色：黑色；内存：256G" |
| 价格 | 显示商品售价，点击可编辑(需二次确认) |
| 库存 | 显示当前库存，红色警示库存<10的商品，点击可编辑库存 |
| 销量 | 显示累计销量 |
| 状态标签 | 上架(绿色)、下架(灰色)、待审核(黄色)、违规(红色) |
| 操作按钮 | 上架/下架(切换状态)、编辑、删除(需确认) |
| 分页 | 底部显示总条数和分页器，每页显示20/50/100条可切换 |

### 3.2 发布商品

#### 3.2.1 选择类目

| 元素 | 交互行为 |
|------|----------|
| 类目选择器 | 树形结构展示一二三级类目，点击选中一级，二级展开，点击选中二级，三级展开，点击选中三级 |
| 最近使用 | 左侧显示最近使用的类目，点击快速选择 |
| 类目确认 | 选中三级类目后显示"已选类目：手机 > 智能手机 > 5G手机"，点击"下一步" |

#### 3.2.2 商品基本信息

| 元素 | 交互行为 |
|------|----------|
| 商品标题 | 输入框，最多60字符，实时显示剩余字符数，超出红色提示 |
| 商品卖点 | 输入商品卖点/副标题，最多50字符 |
| 商品品牌 | 下拉选择品牌，可搜索，支持模糊搜索 |
| 商品属性 | 根据类目动态显示属性项(颜色、尺寸、内存等)，必填项标注*，选择后生效 |
| 属性备注 | 某些属性可填写自定义备注 |

#### 3.2.3 SKU规格设置

| 元素 | 交互行为 |
|------|----------|
| 规格模板 | 可选择已有规格模板或新建规格 |
| 添加规格 | 点击"+ 添加规格"，输入规格名称(如"颜色"、"尺寸")，添加规格值 |
| 颜色选择 | 颜色类规格可选择预设色块或自定义颜色 |
| 规格图片 | 每个规格值可上传对应图片(最多5张)，第一张为主图 |
| SKU生成 | 设置完规格后点击"生成SKU"自动生成SKU列表 |
| SKU编辑 | 表格形式展示每个SKU：规格组合、价格(必填)、库存(必填)、SKU编码、条形码、图片 |
| 批量设置 | 可选择多个SKU批量设置价格/库存 |
| 库存模式 | 选择"拍下减库存"或"付款减库存" |
| SKU删除 | 点击"×"删除单个SKU |

#### 3.2.4 商品详情

| 元素 | 交互行为 |
|------|----------|
| 电脑端详情 | 富文本编辑器，支持图片上传(拖拽/点击上传)、文字格式、表格、视频插入 |
| 图片上传 | 点击上传图片，支持拖拽排序，点击图片可编辑alt文字、链接 |
| 移动端详情 | 可单独设置移动端详情或同步PC端 |
| 详情预览 | 点击"预览"可预览PC端和移动端展示效果 |

#### 3.2.5 商品图片

| 元素 | 交互行为 |
|------|----------|
| 主图上传 | 最多5张，第一张为主图，点击上传，支持拖拽排序 |
| 主图要求 | 显示图片要求提示(尺寸、格式、大小) |
| 视频上传 | 可上传商品视频(可选)，显示视频封面，支持预览 |
| 图片验证 | 上传后自动验证图片是否符合要求，不合格提示原因 |

#### 3.2.6 商品属性

| 元素 | 交互行为 |
|------|----------|
| 基础属性 | 根据类目显示必填和选填属性：上市时间、产地、型号等 |
| 特殊属性 | 如手机显示"CPU型号"、"电池容量"等 |
| 属性值输入 | 下拉选择或手动输入，输入时自动联想 |
| 单位设置 | 数量单位、重量单位等 |

#### 3.2.7 物流与售后

| 元素 | 交互行为 |
|------|----------|
| 运费模板 | 下拉选择运费模板，支持按件/按重量/按体积计算 |
| 运费模板新建 | 点击"新建运费模板"，设置发货地区、运送方式、首件/续件费用 |
| 货到付款 | 开启/关闭货到付款功能 |
| 预售设置 | 开启预售，设置预售发货时间 |
| 七天无理由 | 开启/关闭七天无理由退货，显示详细说明 |
| 质保年限 | 选择质保年限 |

#### 3.2.8 商品上架

| 元素 | 交互行为 |
|------|----------|
| 发布方式 | 选择"立即上架"或"放入仓库" |
| 定时上架 | 可设置定时上架时间，到时自动上架 |
| 提交审核 | 点击"提交审核"，系统先校验必填项，校验通过后提交等待平台审核 |
| 保存草稿 | 点击"保存草稿"，商品保存为草稿状态 |
| 预览 | 点击"预览"查看商品详情页展示效果 |

### 3.3 商品编辑

| 元素 | 交互行为 |
|------|----------|
| 信息修改 | 编辑商品基本信息、SKU信息、价格、库存等 |
| 保存 | 修改后点击"保存"，若是待审核商品需重新提交审核 |
| 上架/下架 | 编辑后可直接操作上下架 |
| 删除 | 点击删除，弹出确认弹窗，确认后删除 |

### 3.4 商品规格管理

| 元素 | 交互行为 |
|------|----------|
| 规格模板列表 | 显示所有规格模板：模板名称、规格数量、使用商品数 |
| 新建规格 | 点击"新建规格"，输入规格名称和规格值 |
| 编辑规格 | 点击编辑，进入规格编辑页，修改后保存 |
| 删除规格 | 点击删除，需确认，提示是否被商品使用 |
| 设为常用 | 可将常用规格设为常用模板 |

## 四、商家后台 - 订单管理

### 4.1 订单列表

#### 交互说明

| 元素 | 交互行为 |
|------|----------|
| 订单搜索 | 支持订单编号、买家昵称、收货人姓名/电话搜索 |
| 时间筛选 | 选择下单时间范围，支持快捷选择：今日/昨日/近7天/近30天/自定义 |
| 订单状态 | Tab切换：全部/待付款/待发货/已发货/已完成/已取消 |
| 订单来源 | 下拉筛选：PC端/APP端/小程序/其他 |
| 订单表格 | 显示订单编号、下单时间、商品信息(图片+名称+规格+数量)、买家信息、金额(商品总额+运费+优惠)、状态、操作 |
| 商品信息 | 显示商品主图缩略图、商品名称(超长省略)、SKU规格组合、购买数量 |
| 买家信息 | 显示买家昵称、买家ID |
| 金额信息 | 显示商品总额、运费、优惠金额、实付金额 |
| 订单状态 | 待付款(橙色)、待发货(蓝色)、已发货(紫色)、已完成(绿色)、已取消(灰色) |
| 操作按钮 | 根据状态显示：付款(待付款)、发货(待发货)、查看(所有状态)、取消(待付款) |
| 批量发货 | 勾选多个待发货订单，点击"批量发货"打开批量发货弹窗 |
| 导出订单 | 点击"导出"可导出订单Excel，支持按筛选条件导出 |

### 4.2 订单发货

#### 交互说明

| 元素 | 交互行为 |
|------|----------|
| 选择物流 | 点击"发货"按钮，弹出发货弹窗，选择物流公司 |
| 填写运单号 | 输入运单号，支持扫码录入，支持批量运单号粘贴 |
| 电子面单 | 点击"电子面单"可在线打印面单，需先配置打印机 |
| 批量发货 | 勾选多个订单后点击"批量发货"，表格形式批量填写物流信息 |
| 发货成功 | 发货成功后订单状态变为"已发货"，短信通知买家 |
| 发货异常 | 如运单号格式错误、超过发货时效等显示红色警告 |

### 4.3 订单详情

| 元素 | 交互行为 |
|------|----------|
| 订单状态 | 大字显示当前状态，时间线显示状态流转记录 |
| 订单编号 | 显示订单编号，点击复制 |
| 基本信息 | 下单时间、订单来源、支付方式、支付时间 |
| 买家信息 | 买家昵称、ID、收货人、联系电话、收货地址 |
| 商品信息 | 商品列表显示图片、名称、SKU、单价、数量、小计 |
| 优惠信息 | 显示使用的优惠券、满减活动等优惠明细 |
| 运费信息 | 显示运费金额 |
| 发票信息 | 显示发票类型、抬头、内容 |
| 买家留言 | 显示买家填写的订单备注 |
| 商家备注 | 可点击"添加备注"填写商家内部备注(买家不可见) |
| 操作日志 | 显示订单操作日志：状态变更、操作人、操作时间 |

## 五、商家后台 - 售后管理

### 5.1 售后列表

| 元素 | 交互行为 |
|------|----------|
| 售后类型 | Tab切换：全部/退款/退货退款/换货/仅退款 |
| 售后状态 | Tab切换：全部/待处理/处理中/已完成/已拒绝 |
| 退款状态 | Tab切换：全部/待退款/已退款/退款失败 |
| 搜索 | 支持售后单号、订单编号、买家昵称搜索 |

| 售后状态 | 待处理(黄色)、处理中(蓝色)、已完成(绿色)、已拒绝(红色) |
| 操作按钮 | 查看详情、处理、导出 |
| 倒计时 | 待处理售后单显示剩余处理时间，超时标红 |

### 5.2 售后详情

| 元素 | 交互行为 |
|------|----------|
| 售后信息 | 显示售后类型、售后原因、申请时间、详细说明 |
| 买家凭证 | 显示买家上传的凭证图片，点击放大查看 |
| 商品信息 | 显示商品图片、名称、SKU、价格、购买数量 |
| 退款信息 | 显示申请退款金额、实际退款金额(可修改) |
| 收货信息 | 退货退款时显示买家退货的物流信息(运单号、物流公司) |
| 处理操作 | 待处理时显示：同意退款/拒绝(需填写原因)/需要买家补充凭证 |
| 填写退款金额 | 同意后填写退款金额，点击确认退款 |
| 退货地址 | 退货退款时显示平台退货地址，买家需寄回商品 |
| 聊天记录 | 显示与买家的聊天记录 |
| 处理日志 | 显示售后处理日志 |

### 5.3 处理售后

| 元素 | 交互行为 |
|------|----------|
| 同意退款 | 点击后填写退款金额(默认申请金额)，点击确认退款 |
| 拒绝售后 | 点击后弹出原因选择或手动输入拒绝原因，确认拒绝 |
| 等待买家退货 | 同意退货退款后，等待买家填写退货物流信息 |
| 确认收货 | 收到买家退货后，点击"确认收货"完成退款 |
| 换货处理 | 买家寄回商品后，点击"确认收货"并填写换货物流信息 |
| 批量处理 | 可勾选多个售后单，批量同意或拒绝 |

## 六、商家后台 - 客服管理

### 6.1 消息中心

| 隐藏评价 | 违规评价可点击"隐藏" |

## 七、商家后台 - 财务管理

### 7.1 货款结算

| 元素 | 交互行为 |
|------|----------|
| 结算周期 | 显示当前结算周期(如3月16日-3月31日) |
| 账户余额 | 显示可提现余额，点击"提现" |
