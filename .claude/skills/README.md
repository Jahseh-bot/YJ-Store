# Progress Skill - YJ商城开发进度追踪

## 功能
跟踪YJ商城的开发进度，标记已完成的任务。

## 使用方法

### 查看进度
```bash
node .claude/skills/progress-tracker.js show
```
或直接告诉我 `/progress show`

### 标记任务完成
```bash
node .claude/skills/progress-tracker.js mark <任务ID>
```
例如：`node .claude/skills/progress-tracker.js mark 1.1`

告诉我：`/progress mark 1.1`

### 取消完成标记
```bash
node .claude/skills/progress-tracker.js unmark <任务ID>
```

### 重置所有进度
```bash
node .claude/skills/progress-tracker.js reset
```

### 查看所有任务ID
```bash
node .claude/skills/progress-tracker.js list
```
