## 一、产品概述

### 1.1 产品定位

YJ商城是一款面向C端消费者的综合性B2C电商平台，支持PC Web端购物体验，对标淘宝、京东、拼多多等主流电商平台，提供商品浏览、搜索、购买、支付、售后等全链路购物服务。

### 1.2 产品目标

- 为用户提供便捷、安全、丰富的在线购物体验
- 为商家提供高效的商品管理和订单处理能力
- 打造高转化率、高复购率的电商生态

### 1.3 目标用户

| 用户角色 | 描述 |
|---------|------|
| 买家 | 有购物需求的消费者，18-45岁 |
| 卖家/商家 | 在平台开店经营商品的商家 |
| 平台运营 | 负责平台活动、类目管理、审核等 |
| 系统管理员 | 负责系统配置、权限管理等 |

## 四、商品列表页

### 4.1 筛选面板

#### 交互说明

| 元素 | 交互行为 |
|------|----------|
| 品牌筛选 | 点击展开品牌列表(多选)，选中项显示蓝色对勾，右侧显示"已选N个"，点击"×"取消单个，点击"清空"取消全部 |
| 价格区间 | 显示价格滑块或输入框(最低价-最高价)，输入后点击"确定"生效，点击"重置"恢复默认 |
| 属性筛选 | 如颜色、尺寸、内存等，点击展开选项列表，选择后显示已选标签，可删除 |
| 筛选条件标签 | 每个筛选条件显示为标签，点击"×"移除该筛选 |
| 更多筛选 | 点击展开更多筛选条件(品牌、原产地等) |

### 4.2 排序选项

| 排序方式 | 交互行为 |
|----------|----------|
| 综合 | 默认排序，按商品质量分排序 |
| 销量 | 点击按销量从高到低排序，显示红色箭头 |
| 价格 | 点击在升序/降序之间切换，hover放大镜图标显示"点击切换排序" |
| 新品 | 按上架时间排序 |
| 评分 | 按商品评分排序 |

### 4.3 商品卡片

#### 交互说明

| 元素 | 交互行为 |
|------|----------|
| 商品图片 | 默认显示主图，hover时切换为第二张图(若有)，左下角显示"广告"标签(付费推广) |
| 商品标题 | 最多显示2行，超出省略，hover下划线显示完整标题tooltip |
| 价格 | 显示促销价(红色)，划线价(灰色)在下方 |
| 促销标签 | 显示"限时特惠"、"满减"等标签，最多显示2个 |
| 销量 | 显示"已售N件" |
| 收藏按钮 | 点击心形图标，动画变为红色实心，显示"收藏成功"toast |
| 快速加入购物车 | hover时显示半透明购物车按钮，点击直接加入购物车，弹出成功toast |
| 对比框 | 点击商品右上角复选框添加到对比栏，对比栏固定在页面底部 |

### 4.4 分页/加载更多

| 交互行为 |
|----------|
| 滚动加载：滑到底部时显示loading动画，自动加载下一页数据 |
| 手动加载：底部显示"加载更多"按钮，点击加载，loading时按钮变为"加载中..." |
| 返回顶部：滚动超过一屏时显示返回顶部按钮，点击平滑滚动到顶部 |

### 9.1 订单列表

#### Tab切换

| 元素 | 交互行为 |
