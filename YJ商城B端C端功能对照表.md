## 三、发现的问题与解决方案

### 3.1 数据不一致问题

| 问题 | C端描述 | B端描述 | 解决方案 |
|------|---------|---------|----------|
| 发票信息 | C端支持选择电子/纸质发票，可填写纳税人识别号 | B端商家订单详情显示发票信息 | 确认字段一致：发票类型、抬头、内容、纳税人识别号 |
| 收货地址 | C端收货地址包含：收货人、手机号、省市区、详细地址 | B端订单详情显示收货人、联系电话、收货地址 | 字段完全一致 |
| 订单金额计算 | C端显示：商品总价+运费-优惠+积分抵扣=实付 | B端显示：商品总额+运费-优惠金额=实付金额 | 确认积分抵扣逻辑一致 |
| 库存同步 | C端显示实时库存，<10显示"仅剩N件" | B端可编辑库存，<10标红警示 | 库存数据同源同一数据库 |

### 3.2 缺失功能补全

| 缺失项 | 说明 | 建议补充到B端 |
|--------|------|---------------|
| 热搜词配置 | C端搜索框显示热搜词 | 运营后台-系统设置-热门搜索 |
| 分类导航数据 | C端分类导航显示一二三级类目 | 运营后台-类目管理 |
| 帮助中心内容 | C端客户服务-帮助中心 | 运营后台-系统设置-常见问题FAQ |
| 用户协议/隐私政策 | C端注册页展示 | 运营后台-系统设置-用户协议/隐私政策 |
| Banner轮播内容 | C端首页顶部轮播 | 运营后台-广告管理 |
| 快捷入口配置 | C端首页快捷入口 | 运营后台-广告管理或单独配置 |
| 消息通知 | C端应显示系统消息、订单通知等 | 运营后台-客服管理-系统消息发送 |

### 3.3 需明确的数据归属

| 数据项 | 归属 | 说明 |
|--------|------|------|
| 买家收藏夹 | C端私有 | 仅买家可见，B端不展示 |
| 买家足迹 | C端私有 | 仅买家可见，B端不展示 |
| 买家优惠券 | C端私有 | 用户已领取的优惠券，B端可查看统计但不可操作 |
| 商家内部备注 | B端私有 | 订单备注仅商家可见 |
| 平台客服消息 | B端共享 | 运营可查看所有买卖家消息 |

## 四、状态流转对应表

### 4.1 商品状态

```
B端商家操作                    B端运营审核              C端展示
     │                            │                      │
     │ 提交审核 ─────────────────>│                      │
     │                            │  通过 ───────────────>│ 上架
     │                            │  拒绝 ───────────────>│ (商品审核不通过)
     │                            │                      │
     │ 上架 ─────────────────────>│                      │ 上架
     │ 下架 ─────────────────────>│                      │ 下架
     │ 违规 ─────────────────────>│                      │ 违规(商品详情页显示)
```

### 4.2 订单状态

```
C端买家                    B端商家                    C端/B端
   │                         │                          │
   │ 下单(待付款) ─────────────────────────────────────>│ 待付款
   │ 支付 ─────────────────────────────────────────────>│ 待发货
   │ 取消 ─────────────────────────────────────────────>│ 已取消
   │                         │ 发货 ──────────────────>│ 已发货
   │                         │                          │
   │ 确认收货 ─────────────────────────────────────────>│ 已完成
   │ 申请售后 ─────────────────────────────────────────>│ 退款/售后中
```

### 4.3 售后状态

```
C端买家              B端商家                B端运营
   │                   │                      │
   │ 申请售后 ──────────────────────────────────>│ 待处理
   │                   │ 处理中 ───────────────>│
   │                   │ 同意退款 ─────────────>│ 已完成(退款)
   │                   │ 拒绝 ─────────────────>│ 已拒绝
   │                   │                      │
   │ 退货 ────────────>│ 待收货                │
   │                   │ 确认收货 ─────────────>│
```

### 4.4 优惠券状态

```
B端运营                    C端                    状态
   │                        │                      │
   │ 创建优惠券 ────────────────────────────────────>│ 待发放
   │ 发放优惠券 ────────────────────────────────────>│ 待领取/发放中
   │                        │ 领取 ────────────────>│ 待使用
   │                        │ 使用(下单) ──────────>│ 已使用
   │                        │ 过期 ────────────────>│ 已过期
```
